# Harvest Universal Liquidator Polygon

The gateway for all Harvest strategies to execute swaps with multiple Dexes.

## Get Started

1.  Fill in env vars

    ```bash
    cp .env.example .env
    ```

2.  Setup dependencies

    ```bash
    yarn
    ```

### Prerequisites

- [Foundry](https://github.com/foundry-rs/foundry)

### Test locally

```bash
forge test -vvv
```
